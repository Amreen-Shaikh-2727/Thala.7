## Project Overview

Thala.7 is a responsive e-commerce website designed to showcase products with a modern UI. It includes features like product listings, product details page, cart system, and a checkout/payment interface.

The project focuses on frontend development using HTML, CSS, and JavaScript, with emphasis on layout design, responsiveness, and user interaction.

---

## Features

* Responsive design for desktop, tablet, and mobile
* Product listing with categories
* Single product detail page with image gallery
* Add to cart functionality
* Cart page with quantity management
* Price calculation and summary section
* Checkout and payment UI layout
* Navigation bar with menu toggle for mobile
* Clean and modern UI design

---

## Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Media Queries)
* JavaScript (DOM manipulation)

## How to Run the Project

### Option 1: Run Locally

1. Download or clone the repository:

```
git clone https://github.com/amreen-shaikh-2727/Thala.7.git
```

2. Open the project folder

3. Open `index.html` in your browser

---

### Option 2: Live Server (Recommended)

1. Open the project in VS Code
2. Install Live Server extension
3. Right-click on `index.html`
4. Click **"Open with Live Server"**
